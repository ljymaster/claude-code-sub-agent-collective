# Claude Code Collective Framework

This project uses the Claude Code Collective framework for TDD-driven development.

## Framework Documentation

See `.claude-collective/` directory for complete framework documentation:

- **Task System**: @./.claude-collective/task-system.md
- **Agent System**: @./.claude-collective/agents.md
- **Hook System**: @./.claude-collective/hooks.md
- **Quality Standards**: @./.claude-collective/quality.md

## Commands

- `/van` - Start collective workflow with task breakdown and TDD enforcement
- `/van logging enable` - Enable deterministic logging
- `/van logging status` - Check logging status
- `/van logging disable` - Disable logging
